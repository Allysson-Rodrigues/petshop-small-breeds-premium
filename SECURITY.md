# Security Policy

## Reporting

Do not open public issues for vulnerabilities, credential exposure, or auth bypass reports.

Report security findings privately to:

- `contato@allyssonrodrigues.dev`

Include:

- affected route, page, or subsystem
- reproduction steps
- impact assessment
- logs or payload samples when safe to share

## Scope Highlights

Security-sensitive areas in this repository include:

- authentication and session cookies
- admin authorization and dashboard permissions
- public booking request endpoints
- deployment rewrites between frontend and backend
